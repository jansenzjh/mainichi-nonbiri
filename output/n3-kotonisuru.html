【Ｎ３文法】～ことにする／こととする

接続

動詞辞書形＋ことにする／こととする
動詞ない形＋ことにする／こととする

意味

（我）决定…
～することに決める

解説

話し手自身の意志や判断による主観的な決定に用いる文法です。
「～ことにする」は口語としてよく使われ、「～こととする」はやや硬く書き言葉でも使われます。
他者の判断、客観的な決定に使うことはできません。

意志動詞にしか接続できません。

例文

（１）　明日から早起きする ことにしました 。
明天开始早起的决定。
I’ve decided to start waking up early from tomorrow.
（２）　ダイエットを始める ことにした 。
决定开始减肥。
I’ve decided to start a diet.
（３）　毎日の運動を習慣にする ことにした 。
决定把运动变成习惯。
I’ve decided to make exercise a habit.
（４）　部屋を模様替えする ことにした 。
决定重新布置房间。
I’ve decided to rearrange the room.
（５）　自炊をする ことにした 。
决定开始自己做饭。
I’ve decided to start cooking for myself.
（６）　車を購入する ことにしました 。
决定购买一辆车。
I’ve decided to buy a car.
（７）　朝のストレッチを始める ことにしました 。
决定开始早上做伸展运动。
I’ve decided to start doing morning stretches.
（８）　節約生活を始める ことにしました 。
决定开始过节约生活。
I’ve decided to start living frugally.
（９）　自分へのご褒美は来月のボーナスで買う ことにする 。
决定用下个月的奖金来奖励自己。
I’ve decided to reward myself with next month’s bonus.
（10）　明日に向けて風呂に入って寝る ことにする 。
决定洗个澡后早点睡觉。
I’ve decided to take a bath and go to bed early for tomorrow.
（11）　気に入らないこともあるけど、せっかくだから心ゆくまで楽しむ ことにした 。
虽然会有不喜欢的事情，但我决定尽情享受。
Despite there being things I don’t like, I’ve decided to enjoy myself to the fullest.
（12）　拾った子猫を飼う ことにした 。
决定养拾到的小猫。
I’ve decided to adopt the stray kitten.
（13）　地球温暖化防止の為、化石燃料を使う代わりに自然エネルギーを利用する こととした 。
为了防止全球变暖，决定使用自然能源而不是化石燃料。
In order to prevent global warming, I’ve decided to use renewable energy instead of fossil fuels.
（14）　今日からプログラミングの勉強を始める ことにした 。
决定从今天开始学习编程。
I’ve decided to start learning programming from today.

備考

似た文型に「 【Ｎ３文法】～ことになる／こととなる 」があります。
こちらは他人による判断や客観的な決定、自然の成り行きなどによって物事が決まったときに用いられます。
