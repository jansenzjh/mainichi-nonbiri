接続

可能形＋ものも＋可能否定形
辞書形＋ものも＋否定形
～できるものも＋～できない
～できるものもできない

意味

（～できるはずだったのに）できない

解説

本来できることが何らかの原因によってできなくなったことを表します。
前項には「～たら／～ては／～ば／～なら」がよく接続されます。

例文

（１）　相手は格下だからといっても、コンディションが悪ければ勝てる ものも 勝て ない 。
虽说对方是低级，但如果条件不好的话，能赢的也赢不了。
Just because the opponent is of lower rank, if my condition is bad, there are matches I can’t win.
（２）　最初は騙すつもりで接触したのに、こんなに優しくされては騙せる ものも 騙せ なく なる。
刚开始是打算骗人的，但是被这么温柔的对待，真的不忍心欺骗了。
Even though I initially intended to deceive, if I’m treated with such kindness, there are people I can’t deceive anymore.
（３）　あんな気持ち悪いものを見せられた後なら食べられる ものも 食べられ ない 。
看到那种让人恶心的东西之后就吃不下了。
After being shown something so disgusting, there are foods I can’t eat.
（４）　具合が悪いのにそんな動き回っていては治る ものも 治ら ない だろう。
明明身体不舒服，但是这样到处乱跑的话治得好的病也治不好吧。
If I’m feeling unwell and still running around like that, there are things that won’t heal.
（５）　勉強中に大音量で音楽を流されたら、覚えられる ものも 覚えられ ない だろう。
在学习中如果被大音量播放音乐，连能记住的东西都记不住吧。
If loud music is played while studying, there are things I won’t be able to remember.
（６）　貯金すると言いつつあれもこれもと欲しいものを欲しいだけ買っていたら貯まる ものも 貯まら ない はずだ。
一边说要存钱，一边想买想要的东西的话，应该不会存钱的。
If I keep buying all the things I desire while claiming to save money, there are things that won’t accumulate.
（７）　集中力を欠いていたらできる ものも でき ない 。
如果缺乏集中力，不能做到。
If I lack concentration, there are things I can’t accomplish.
（８）　こんなに寒くては指が動かず、演奏できる ものも でき ない 。
这么冷手指无法动弹，无法演奏。
It’s so cold that my fingers won’t move, and there are things I can’t play.

備考

【Ｎ０文法】～れないものは～れない／～できないものは～できない 」
